var, let, const

These keywords are used to declare variable in JavaScript. They have different rules for scope, reassignment and hoisting. var is the function scoped means that the variable can only be accessed within the function where it is declared, and not outside of it. let and const is the block scoped means that the variable can only be accessed within the block where it is declared, and not outside of it. A block is a section of code enclosed by curly braces {...}

null and undefined

null means there is no value. It is used to represent empty. undefined means a variable has not been assigned a value or a property does not exist on an object.

REST API

A representation is the way a resource is presented to the client, such as JSON, XML, HTML, etc. A REST API allows the client to interact with the resources on the server by using standard HTTP methods, such as GET, POST, PUT, DELETE, etc. Each method has a specific meaning and purpose for manipulating the resources. A REST API is one of the most common and popular ways of building web-based APIs, as it offers many benefits, such as simplicity, scalability, interoperability, and extensibility.
